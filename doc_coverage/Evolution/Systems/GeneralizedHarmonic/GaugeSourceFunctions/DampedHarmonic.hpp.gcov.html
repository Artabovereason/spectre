      49             :  * \brief Damped harmonic gauge source function.
      50             :  *
      51             :  * \details Computes the gauge source function designed for binary black hole
      52             :  * evolutions. These have been taken from \cite Szilagyi2009qz and
      53             :  * \cite Deppe2018uye.
      54             :  *
      55             :  * The covariant form of the source function \f$H_a\f$ is written as:
      56             :  *
      57             :  * \f{align*}
      58             :  * H_a :=  [1 - R_{H_\mathrm{init}}(t)] H_a^\mathrm{init} +
      59             :  *  [\mu_{L1} \mathrm{log}(\sqrt{g}/N) + \mu_{L2} \mathrm{log}(1/N)] t_a
      60             :  *   - \mu_S g_{ai} N^i / N
      61             :  * \f}
      62             :  *
      63             :  * where \f$N, N^k\f$ are the lapse and shift respectively, \f$t_a\f$ is the
      64             :  * unit normal one-form to the spatial slice, and \f$g_{ab}\f$ is
      65             :  * the spatial metric (obtained by projecting the spacetime metric onto the
      66             :  * 3-slice, i.e. \f$g_{ab} = \psi_{ab} + t_a t_b\f$). The prefactors are:
      67             :  *
      68             :  * \f{align*}
      69             :  *  \mu_{L1} &= A_{L1} R_{L1}(t) W(x^i) \mathrm{log}(\sqrt{g}/N)^{e_{L1}}, \\
      70             :  *  \mu_{L2} &= A_{L2} R_{L2}(t) W(x^i) \mathrm{log}(1/N)^{e_{L2}}, \\
      71             :  *  \mu_{S} &= A_{S} R_{S}(t) W(x^i) \mathrm{log}(\sqrt{g}/N)^{e_{S}},
      72             :  * \f}
      73             :  *
      74             :  * temporal roll-on functions \f$ R_X(t)\f$ (for \f$ X = \{L1, L2, S\} \f$) are:
      75             :  *
      76             :  * \f{align*}
      77             :  * \begin{array}{ll}
      78             :  *     R_X(t) & = 0, & t< t_{0,X} \\
      79             :  *             & = 1 - \exp[-((t - t_{0,X})/ \sigma_t^X)^4], & t\geq t_{0,X} \\
      80             :  * \end{array}
      81             :  * \f}
      82             :  *
      83             :  * and the spatial weight function is:
      84             :  *
      85             :  * \f{align*}
      86             :  * W(x^i) = \exp[-(r/\sigma_r)^2].
      87             :  * \f}
      88             :  * This weight function can be written with multiple constant factors in the
      89             :  * exponent in literature \cite Deppe2018uye, but we absorb them all into
      90             :  * \f$ \sigma_r\f$ here. The coordinate \f$ r\f$ is the Euclidean radius
      91             :  * in Inertial coordinates.
      92             :  *
      93             :  * Note that for the last three terms in \f$H_a\f$ (with \f$ X = \{L1, L2, S\}
      94             :  * \f$):
      95             :  *   - Amplitude factors \f$ A_{X} \f$ are taken as input here as `amp_coef_X`
      96             :  *   - Roll-on factor \f$ R_{X} \f$ is specified completely by \f$\{ t_{0,X},
      97             :  *     \sigma_t^X\}\f$, which are taken as input here as `t_start_X` and
      98             :  *     `sigma_t_X`.
      99             :  *   - Exponents \f$ e_X\f$ are taken as input here as `exp_X`.
     100             :  *   - Spatial weight function \f$W\f$ is specified completely by
     101             :  *     \f$\sigma_r\f$, which is taken as input here as `sigma_r`.
     102             :  */
