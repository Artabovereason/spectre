     147             :  * \brief Computes the upwind flux for scalar-waves in curved spacetime.
     148             :  *
     149             :  * \details The upwind flux is given by \cite Teukolsky2015ega :
     150             :  *
     151             :  * \f[
     152             :  * G = S\left(\Lambda^+ S^{-1} U^{\rm int}
     153             :  *           + \Lambda^- S^{-1} U^{\rm ext}\right)
     154             :  * = S \left(\Lambda^+ \hat{U}^{\rm int}
     155             :  *           + \Lambda^- \hat{U}^{\rm ext}\right),
     156             :  * \f]
     157             :  *
     158             :  * where
     159             :  *
     160             :  *  - \f$G\f$ is the numerical upwind flux dotted with the interface normal;
     161             :  *  - \f$U\f$ is a vector of all evolved variables;
     162             :  *  - \f$S\f$ is a matrix whose columns are the eigenvectors of the
     163             :  *       characteristic matrix for the evolution system. It maps the
     164             :  *       evolved variables to characteristic variables \f$\hat{U}\f$, s.t.
     165             :  *       \f$\hat{U} := S^{-1}\cdot U\f$; and
     166             :  *  - \f$\Lambda^\pm\f$ are diagonal matrices containing
     167             :  *       positive / negative eigenvalues of the same matrix as its elements.
     168             :  *
     169             :  * The superscripts \f${\rm int}\f$ and \f${\rm ext}\f$ on \f$U\f$ indicate
     170             :  * that the corresponding set of variables at the element interface have been
     171             :  * taken from the _interior_ or _exterior_ of the element. Exterior of the
     172             :  * element is naturally the interior of its neighboring element. Therefore,
     173             :  * \f$\hat{U}^{\rm int}:= S^{-1}U^{\rm int}\f$ are the characteristic variables
     174             :  * at the element interface computed using evolved variables from the interior
     175             :  * of the element, and \f$\hat{U}^{\rm ext}= S^{-1}U^{\rm ext}\f$ are the
     176             :  * same computed from evolved variables taken from the element exterior, i.e.
     177             :  * the neighboring element.
     178             :  * The sign of characteristic speeds indicates the direction of propagation of
     179             :  * the corresponding characteristic field with respect to the interface normal
     180             :  * that the field has been computed along (with negative speeds indicating
     181             :  * incoming characteristics, and with positive speeds indicating outgoing
     182             :  * characteristics). Therefore, \f$\Lambda^+\f$ contains characterstic speeds
     183             :  * for variables that are outgoing from the element at the interface, and
     184             :  * \f$\Lambda^-\f$ contains characteristic speeds for variables that are
     185             :  * incoming to the element at the interface. An ambiguity naturally arises
     186             :  * as to which set of evolved variables (''interior'' or ''exterior'') to use
     187             :  * when computing these speeds. Following the treatment of generalized harmonic
     188             :  * evolution system's characteristics in its upwind flux (\ref
     189             :  * GeneralizedHarmonic::UpwindFlux ), here also we compute both and use their
     190             :  * average, i.e. \f$\lambda^{\rm avg} = (\lambda^{\rm int} + \lambda^{\rm
     191             :  * ext})/2\f$, to populate \f$\Lambda^\pm\f$.
     192             :  *
     193             :  *
     194             :  * This function computes the upwind flux as follows:
     195             :  *  -# Computes internal and external characteristic variables and speeds using
     196             :  * evolved variables from
     197             :  *     both the element interior and exterior.
     198             :  *  -# Computes the average characteristic speeds and constructs
     199             :  *      \f$\Lambda^{\pm}\f$.
     200             :  *  -# Computes the upwind flux as a weigted sum of external and internal
     201             :  *     characteristic variables
     202             :  *
     203             :  * \f[
     204             :  * G = S\left(w_{\rm ext} \hat{U}^{\rm ext}
     205             :  *       + w_{\rm int} \hat{U}^{\rm int}\right),
     206             :  * \f]
     207             :  *
     208             :  * with weights \f$w_{\rm ext} = \Theta(-\Lambda)\cdot\Lambda\f$, and
     209             :  * \f$w_{\rm int} = \Theta(\Lambda)\cdot\Lambda\f$, where \f$\Theta\f$ is the
     210             :  * step function centered at zero, \f$\Lambda = \Lambda^+ + \Lambda^-\f$, and
     211             :  * the dot operator \f$(\cdot)\f$ indicates an element-wise product.
     212             :  *
     213             :  * \warning With the averaging of characteristic speeds, this flux does not
     214             :  * satisfy the generalized Rankine-Hugoniot conditions
     215             :  * \f${G}^{\rm int}(\hat{U}^{\rm int}, \hat{U}^{\rm ext})
     216             :  * = - {G}^{\rm ext}(\hat{U}^{\rm ext}, \hat{U}^{\rm int})\f$,

     218             :  * entering the neighboring element, and vice versa. This condition is
     219             :  * important for a well-balanced scheme, and so please use this flux with
     220             :  * caution.
     221             :  */
