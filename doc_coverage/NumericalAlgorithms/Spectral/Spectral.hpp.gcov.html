     262             : /*!
     263             :  * \brief %Matrix used to perform an indefinite integral of a function over the
     264             :  * logical grid. The left boundary condition is such that the integral is 0 at
     265             :  * \f$\xi=-1\f$
     266             :  *
     267             :  * Currently only Legendre and Chebyshev polynomials are implemented, but we
     268             :  * provide a derivation for how to compute the indefinite integration matrix for
     269             :  * general Jacobi polynomials.
     270             :  *
     271             :  * #### Legendre Polynomials
     272             :  * The Legendre polynomials have the identity:
     273             :  *
     274             :  * \f{align*}{
     275             :  * P_n(x) = \frac{1}{2n+1}\frac{d}{dx}\left(P_{n+1}(x) - P_{n-1}(x)\right)
     276             :  * \f}
     277             :  *
     278             :  * The goal is to evaluate the integral of a function \f$u\f$ expanded in terms
     279             :  * of Legendre polynomials as
     280             :  *
     281             :  * \f{align*}{
     282             :  * u(x) = \sum_{i=0}^{N} c_i P_i(x)
     283             :  * \f}
     284             :  *
     285             :  * We similarly expand the indefinite integral of \f$u\f$ as
     286             :  *
     287             :  * \f{align*}{
     288             :  * \left.\int u(y) dy\right\rvert_{x}=&\sum_{i=0}^N \tilde{c}_i P_i(x) \\
     289             :  *   =&\left.\int\sum_{i=1}^{N}\frac{c_i}{2i+1}
     290             :  *      \left(P_{i+1}(y)-P_{i-1}(y)\right)dy\right\rvert_{x}
     291             :  *      + \tilde{c}_0 P_0(x) \\
     292             :  *   =&\sum_{i=1}^{N}\left(\frac{c_{i-1}}{2i-1} - \frac{c_{i+1}}{2i+3}\right)
     293             :  *     P_i(x) + \tilde{c}_0 P_0(x)
     294             :  * \f}
     295             :  *
     296             :  * Thus we get that for \f$i>0\f$
     297             :  *
     298             :  * \f{align*}{
     299             :  * \tilde{c}_i=\frac{c_{i-1}}{2i-1}-\frac{c_{i+1}}{2i+3}
     300             :  * \f}
     301             :  *
     302             :  * and \f$\tilde{c}_0\f$ is a constant of integration, which we choose such that
     303             :  * the integral is 0 at the left boundary of the domain (\f$x=-1\f$). The
     304             :  * condition for this is:
     305             :  *
     306             :  * \f{align*}{
     307             :  *   \tilde{c}_0=\sum_{i=1}^{N}(-1)^{i+1}\tilde{c}_i
     308             :  * \f}
     309             :  *
     310             :  * The matrix returned by this function is the product of the tridiagonal matrix
     311             :  * for the \f$\tilde{c}_i\f$ and the matrix for the boundary condition.
     312             :  *
     313             :  * #### Chebyshev Polynomials
     314             :  *
     315             :  * A similar derivation leads to the relations:
     316             :  *
     317             :  * \f{align*}{
     318             :  *  \tilde{c}_i=&\frac{c_{i-1}-c_{i+1}}{2i},&\mathrm{if}\;i>1 \\
     319             :  *  \tilde{c}_1=&c_0 - \frac{c_2}{2},&\mathrm{if}\;i=1 \\
     320             :  * \f}
     321             :  *
     322             :  * We again have:
     323             :  *
     324             :  * \f{align*}{
     325             :  * \tilde{c}_0=\sum_{i=1}^N(-1)^{i+1}\tilde{c}_i
     326             :  * \f}
     327             :  *
     328             :  * These are then used to define the indefinite integration matrix.
     329             :  *
     330             :  * #### Jacobi Polynomials
     331             :  *
     332             :  * For general Jacobi polynomials \f$P^{(\alpha,\beta)}_n(x)\f$ given by
     333             :  *
     334             :  * \f{align*}{
     335             :  *  (1-x)^\alpha(1+x)^\beta P^{(\alpha,\beta)}_n(x)=\frac{(-1)^n}{2^n n!}
     336             :  *  \frac{d^n}{dx^n}\left[(1-x)^{\alpha+n}(1+x)^{\beta+n}\right]
     337             :  * \f}
     338             :  *
     339             :  * we have that
     340             :  *
     341             :  * \f{align*}{
     342             :  * P^{(\alpha,\beta)}_n(x)=\frac{d}{dx}\left(
     343             :  * b^{(\alpha,\beta)}_{n-1,n}P^{(\alpha,\beta)}_{n-1}(x) +
     344             :  * b^{(\alpha,\beta)}_{n,n}P^{(\alpha,\beta)}_n(x) +
     345             :  * b^{(\alpha,\beta)}_{n+1,n}P^{(\alpha,\beta)}_{n+1}(x)
     346             :  * \right)
     347             :  * \f}
     348             :  *
     349             :  * where
     350             :  *
     351             :  * \f{align*}{
     352             :  * b^{(\alpha,\beta)}_{n-1,n}=&-\frac{1}{n+\alpha+\beta}
     353             :  *                            a^{(\alpha,\beta)}_{n-1,n} \\
     354             :  * b^{(\alpha,\beta)}_{n,n}=&-\frac{2}{\alpha+\beta}
     355             :  *                          a^{(\alpha,\beta)}_{n,n} \\
     356             :  * b^{(\alpha,\beta)}_{n+1,n}=&\frac{1}{n+1}
     357             :  *                            a^{(\alpha,\beta)}_{n+1,n} \\
     358             :  * a^{(\alpha,\beta)}_{n-1,n}=&\frac{2(n+\alpha)(n+\beta)}

     366             :  * Following the same derivation we get that

     455             :  * analytically for Gauss quadrature by evaluating

     457             :  * \frac{(u,\Phi_i)}{\gamma_i}\f$

     459             :  * \f$\mathcal{V}^{-1}_{ij}=\mathcal{V}_{ji}\frac{w_j}{\gamma_i}\f$ where the
     460             :  * \f$w_j\f$ are the Gauss quadrature weights and \f$\gamma_i\f$ is the norm
     461             :  * square of the spectral basis function \f$\Phi_i\f$.
